== Description ==
Is your WooCommerce shop URLs SEO friendly? Are you taking advantage of all optimization techniques to maximize the visibility of your online store to search engines? You can create SEO Friendly Product URLs
using "SEO-friendly URLs for WooCommerce" - the most advanced and easy to use permalink manager plugin for WooCommerce.

== Installation ==
	= Using The WordPress Dashboard =
	* Navigate to the 'Add New' in the plugins dashboard
	* Search for SEO-friendly URLs for WooCommerce
	* Click Install Now
	* Activate the plugin on the Plugin dashboard

	= Uploading in WordPress Dashboard =
	* Navigate to the 'Add New' in the plugins dashboard
	* Navigate to the 'Upload' area
	* Select seo-friendly-urls-for-woocommerce.zip from your computer
	* Click 'Install Now'
	* Activate the plugin in the Plugin dashboard

	= Using FTP =
	* Download seo-friendly-urls-for-woocommerce.zip
	* Extract the seo-friendly-urls-for-woocommerce directory to your computer
	* Upload the seo-friendly-urls-for-woocommerce directory to the /wp-content/plugins/ directory
	* Activate the plugin in the Plugin dashboard

	= Setting Options =
	* Setting Page is located inside WooCommerce Tabs named as Permalinks
